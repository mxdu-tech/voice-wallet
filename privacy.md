# Privacy Policy for Voice Wallet

Last updated: April 2026

## Overview

Voice Wallet is a browser extension that allows users to create, import, and manage an Ethereum wallet locally, and execute blockchain transactions using natural language commands.

We respect your privacy. This extension is designed to operate primarily on the user's device and does not collect or transmit personal data to external servers.

---

## Data Collection

Voice Wallet may process the following data locally on your device:

- Encrypted wallet data (private keys, mnemonic phrases)
- User preferences
- Transaction inputs (e.g., recipient address, amount)

All sensitive data is encrypted before being stored using browser storage.

---

## Data Usage

The data is used solely for:

- Wallet management
- Transaction creation and signing
- Improving user experience (e.g., session state)

---

## Data Storage

- All sensitive data is stored locally in the browser using encrypted storage.
- Private keys and mnemonic phrases never leave the user's device.

---

## Data Sharing

Voice Wallet does NOT:

- Sell user data
- Share personal data with third parties

However, the extension interacts with public blockchain RPC endpoints (e.g., Ethereum nodes) to:

- Fetch blockchain data
- Broadcast signed transactions

These interactions may expose your public wallet address and transaction data, which are inherently public on the blockchain.

---

## Remote Code

Voice Wallet does not use remote code. All logic is packaged within the extension.

---

## Security

We implement encryption and local storage mechanisms to protect sensitive data. However, users are responsible for safeguarding their recovery phrases and passwords.

---

We may update this Privacy Policy from time to time. Updates will be reflected on this page.
